上海阿姨将小区草坪改成菜园

4月23日，上海几位阿姨将小区草坪改造成了菜园，自己动手用锄头锄地种菜。目击者表示，特殊时期可以理解，疫情过后物业会进行管理。

中国航天日：展望未来的“星辰大海”

“探索浩瀚宇宙，发展航天事业，建设航天强国，是我们不懈追求的航天梦。”几十年来，从“东方红”唱响到“神舟”逐梦，从“北斗”指路到“嫦娥”揽月、从“祝融”探火到“天宫”遨游，中国航天人以一次又一次的非凡突破，让中国印记闪耀星空。今天，是第七个中国航天日，人民网整理了我国计划及酝酿中的航天任务，一同展望中国航天未来的“星辰大海”。

人类为什么要坚持探索宇宙，生活在地球不好吗？背后的原因很现实

“宇宙很大”，目前人类可以观察的宇宙被称为“可观测宇宙”，可观测宇宙的半径大约为465亿光年，在可观测宇宙之外存在着更加广阔的宇宙空间，对于人类来说465亿光年是一个可望而不可及的遥远距离，很多科学家认为，人类文明可能直到灭亡都无法走出银河系。
